Unfair Wheel - Spin the Wheel

Customize your wheel and spin away with or without cheats!

Features

  • Adjust the weight/ratio of each element
  • Change the font color and wheel color of elements
  • Change the title color and background color
  • Adjust the font size of elements
  • Adjust the font size of the title
  • Choose from over 100 sound effects
  • Add your own sound files
  • Add unlimited elements to the wheel
  • Save unlimited data
  • Get random results every time (while Cheat Mode is off)
  • Get desired results every time (while Cheat Mode is on)
  • Swipe the wheel to spin
  • Press the center icon to spin
  • Tap to stop the wheel from spinning
  • Spinning speed changes based on swipe speed
  • Input multiple elements at once using line breaks
  • Export and import data in text format
  • Repeat elements multiple times
  • Change the arrow icon
  • Start with the last wheel you used
  • Hide the tab view by pressing the arrow on the top right of the tab view

Cheat Modes

Mode 1: Swipe the area you want to win.

Mode 2: Release your finger as soon as the arrow points to the desired area.

Mode 3: Increase the cheat weight value of the desired element in the settings, and then just spin the wheel.

How to Play

1. Launch the app and go to the Edit screen.

2. Customize the title and elements as you like.

3. Now you have your very own custom wheel! Press the "SPIN" button on the upper right to play!
